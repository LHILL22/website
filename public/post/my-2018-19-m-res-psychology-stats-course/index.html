My 2018/19 M.Res. Psychology Stats Course

Last updated on Aug 7, 2019 1 min read

As my M.Res. Stats course is slowly morphing into a broader course on reproducibilty, data science, and statistics for psychologists, I thought it would be helpful (possibly for my future self) to add a link here to the 2018-2019 iteration of the course.

All the material can be downloaded from the following repo:

https://github.com/ajstewartlang/Psychology_MRes_Stats_R_Course

In the repo itself, click on the ‘Clone or download’ button in the top right to download the zipped version of the repository.

Each folder contains the slides in .pdf form for the first half of each session, the R script that goes with those slides, and the lab worksheet for the second half as an .html file. All data files for the lectures and the lab classes are included.

Semester 1
Lecture 1 Introduction, Open Science, and Power
Lecture 2 Introduction to R
Lecture 3 Data Wrangling and Visualisation in the Tidyverse
Lecture 4 General Linear Model - Regression
Lecture 5 General Linear Model - Regression
Lecture 7 General Linear Model - ANOVA
Lecture 8 General Linear Model - ANOVA

Semester 2
Workshop - Mixed Models
Workshop - Advanced R (including data simulation)
Workshop - Hackathon!
Workshop - Using Binder for Reproducible Computational Environments
